Detailed Description

A list of embeddings to search for struct field. It is also used to store the target vectors for SearchRequest/SubSearchRequest.

Member Function Documentation

◆ AddSparseVector()

Status milvus::EmbeddingList::AddSparseVector ( const nlohmann::json &  vector)

Add a sparse vector to search. We support two patterns of sparse vector:

  1. a json dict like {"1": 0.1, "5": 0.2, "8": 0.15}.
  2. a json dict like {"indices": [1, 5, 8], "values": [0.1, 0.2, 0.15]}.

◆ SetSparseVectors()

Status milvus::EmbeddingList::SetSparseVectors ( const std::vector< nlohmann::json > &  vectors)

Assign sparse vectors to search request. Note: this method will reset the vector list. We support two patterns of sparse vector:

  1. a json dict like {"1": 0.1, "5": 0.2, "8": 0.15}.
  2. a json dict like {"indices": [1, 5, 8], "values": [0.1, 0.2, 0.15]}.
