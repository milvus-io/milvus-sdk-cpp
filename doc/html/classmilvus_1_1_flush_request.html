void SetWaitFlushedMs (int64_t ms)
 Set milliseconds to wait the flush action done.. Default value is 0. If the WaitFlushedMs is zero, the Flush() will call GetFlushState() to check related segments state, until all the segments are flushed to make sure the buffer persisted successfully. If the WaitFlushedMs is larger than zero, the GetFlushState() will break the loop after a certain of time span and return a status saying the process is timeout.
