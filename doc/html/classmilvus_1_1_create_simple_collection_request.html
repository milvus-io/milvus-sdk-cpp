Detailed Description

Used by MilvusClientV2::CreateCollection() to create a simple collection. The simple collection has only two fields: primary field and vector field. The primary field can be INT64 or VARCHAR type. The vector field is FLOAT_VECTOR type, index is AUTOINDEX.
